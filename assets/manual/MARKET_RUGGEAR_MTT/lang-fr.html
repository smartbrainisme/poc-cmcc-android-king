• Présentation du G&T

    Le G&T -Geo & Talking- est une technologie innovante fournie par M.T.T. qui vous permet de communiquer instantanément avec une ou plusieurs personnes par le biais du réseau GPRS / EDGE / 3G / WiFi, et de localiser votre téléphone grâce au module GPS.

  • Se connecter

    Le M.T.T. SmartMax est équipé d’une touche de raccourci pour le G&T : même lorsque le SmartMax est en veille, un simple appui sur cette touche vous permet d’accéder à la fonction G&T. Chaque SmartMax dispose d’un compte G&T unique. Vous pouvez vous connecter en saisissant votre identifiant et votre mot de passe (par défaut : 123456).

  • Connexion automatique

    Lorsque vous quittez la fonction G&T, vous pouvez cocher « Connexion automatique » pour vous connecter automatiquement la prochaine fois. Le G&T permet également une connexion automatique même lorsque votre compte est déconnecté : si vous recevez un appel, le SmartMax se connectera automatiquement en mode G&T en arrière-plan, sans pour autant recevoir une invitation pour cet appel. Pour les futurs appels, vous recevrez une invitation d’appel.

  • Conversation instantanée au sein d’un groupe

    Une fois la connexion effectuée, cliquez sur pour afficher la liste des groupes générés par le Centre de Gestion du G&T. Sélectionnez le groupe auquel vous souhaitez vous connecter afin d’afficher tous les contacts présents dans le groupe (connectés ou non). Si la conversation est « Inoccupée », maintenez la touche G&T enfoncée ; dès que vous entendrez un « bip », vous pourrez commencer à parler. Si la conversation est occupée par l’un des contacts, vous pouvez tout de même appuyer sur la touche G&T pour être mis automatiquement en file d’attente (sous réserve d’avoir activé cette fonction dans le Centre de Gestion), et vous pourrez parler dès que les contacts auront terminé. Pour rester dans la file d’attente, maintenez la touche G&T enfoncée. Pour quitter la file d’attente, relâchez simplement la touche G&T.
    Vous pouvez basculer entre le mode Combiné et Haut-parleur en appuyant sur ou sur la touche .

  • Message texte/vocal

    Si les contacts déconnectés ne parviennent pas à recevoir vos messages instantanés, vous avez la possibilité de leur laisser un message texte/vocal : dès que vos contacts se connecteront, ils recevront vos messages. Appuyez sur ou faites défiler l’écran vers la gauche pour basculer en mode de message texte/vocal. Appuyez sur pour insérer une photo, un smiley ou bien pour saisir un message texte en quelques clics dans le cadre de saisie ; ou, enfin, cliquez sur pour laisser un message vocal. Sur cette page, vous pourrez consulter tous les messages vocaux instantanés et les messages texte/vocaux que vous avez reçus. Pour cela, cliquez sur pour réécouter les messages.

  • Conversation ponctuelle avec un ou plusieurs contacts

    Vous pouvez créer une conversation ponctuelle avec un ou plusieurs contacts qui se trouvent dans le même groupe : cochez les contacts avec lesquels vous souhaiter parler puis appuyez sur « Appeler » pour les inviter à rejoindre la conversation. Pendant la conversation, vous pouvez inviter d’autres contacts: appuyez sur , puis cochez les contacts à inviter et enfin, appuyez sur pour valider.

  • Appeler le Centre de Gestion

    Vous pouvez appeler le Centre de Gestion en cliquant sur « Centre de Gestion », placé en première place dans votre liste de contacts.

  • Localisation du téléphone

    Le SmartMax est équipé d’un module GPS de localisation de téléphone. Dans « Paramètres de localisation », vous pouvez activer ou désactiver le module GPS et paramétrer la fréquence de synchronisation avec le Centre de Gestion.

  • Envoyer une photo au Centre de Gestion

    Prenez une photo ou sélectionnez une image de l’Album pour l’envoyer au Centre de Gestion.

  • Paramètres

    Pour régler le volume, modifier le nom et le mot de passe.
